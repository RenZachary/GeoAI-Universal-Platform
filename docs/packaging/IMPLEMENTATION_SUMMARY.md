## 📊 打包流程

```
准备阶段
  ↓
清理旧包 → 创建目录结构
  ↓
构建阶段
  ↓
构建后端 → 构建前端
  ↓
捆绑阶段
  ↓
下载Node.js → 提取运行时
  ↓
复制阶段
  ↓
复制后端 → 复制前端 → 复制工作区 → 复制依赖
  ↓
生成阶段
  ↓
创建启动脚本 → 设置环境变量
  ↓
完成
  ↓
验证结果 → 显示包大小
```

## 🔧 技术栈

- **脚本语言**: JavaScript (ES Modules)
- **文件系统**: fs-extra
- **子进程**: child_process
- **网络**: https (下载Node.js)
- **路径处理**: path, fileURLToPath
